## 프로젝트 배경 
- 로또 기본 개념 학습
로또의 기본 원리와 규칙을 쉽게 이해 가능한 프로그램      
초보자도 빠르게 로또 게임의 구조를 파악게 도움을 줌   

- 로또 구매 경험 제공
실제 로또를 구매하기 전, 가상의 로또를 통해 경험을 쌓는 환경 제공      
사용자는 다양한 번호 조합을 시도하며, 로또의 확률과 결과를 실시간으로 체험 가능
